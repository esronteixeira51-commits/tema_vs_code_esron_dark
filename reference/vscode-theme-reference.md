# Documentação oficial do VS Code

Links de referência pra quem for estender o tema — a documentação da Microsoft é a fonte final sobre qualquer chave/scope/token, mais completa do que o resumo dado em `reference/colors.md`, `reference/semantic-tokens.md` e `reference/textmate-scopes.md`.

- **Theme Color Reference** (lista completa de todas as chaves de `colors`): https://code.visualstudio.com/api/references/theme-color
- **Color Theme Guide** (como um tema é estruturado): https://code.visualstudio.com/api/extension-guides/color-theme
- **Semantic Highlighting Guide** (lista de token types e modifiers padrão): https://code.visualstudio.com/api/language-extensions/semantic-highlight-guide
- **TextMate Scope Naming Conventions**: https://www.sublimetext.com/docs/scope_naming.html *(referência histórica, usada pela maioria das gramáticas do VS Code)*

> Esses links podem mudar de endereço com o tempo — se algum estiver quebrado, busque pelo título exato no site da Microsoft.
