# Mapa de sintaxe — Esron Dark

Keyword (controla bloco: `if`, `for`, `class`, `fn`...)
↓
Azul escuro, negrito

Variable
↓
Vermelho

Variable (readonly / const / static readonly)
↓
Vermelho, negrito

Number
↓
Amarelo

String
↓
Verde

Function (usuário — `somar`, `getNome`)
↓
Azul (médio, sem negrito)

Function (biblioteca/utilitário — `printf`, `console.log`, `Math.sqrt`)
↓
Azul claro

Comment
↓
Cinza, itálico

Boolean / Macro / Enum member / Label / Event
↓
Laranja

Decorator / Anotação (`@Override`, `#[derive]`, `[Obsolete]`)
↓
Laranja, itálico

Primitive Type (`int`, `char`, `double`, `bool`, `i32`...)
↓
Roxo

### Fora do sistema de 8 cores (estrutural)

Type definido pelo usuário (`class`, `struct`, `enum`, `interface`)
↓
Salmão

Namespace / módulo
↓
Azul acinzentado (frost blue)

Operador
↓
Cinza-azulado neutro

Tag HTML/JSX
↓
Azul-ciano

Atributo HTML/JSX (e seletor de classe/ID/pseudo-classe em CSS)

Roxo-rosado
