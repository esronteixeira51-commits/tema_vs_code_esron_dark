# Changelog

Todas as mudanças notáveis deste tema são documentadas aqui.

## [0.1.0] — 2026-07-19

### Adicionado
- Primeira versão pública do tema Esron Dark.
- Sistema fechado de 8 cores semânticas (azul escuro, vermelho, amarelo, verde, azul claro, cinza, laranja, roxo), com negrito e itálico usados como segunda dimensão de codificação visual.
- Cobertura de semantic tokens e fallback via TextMate scopes, testada manualmente em C++, Java, C#, Go, Rust, CSS, HTML, JavaScript e XML.
- Paleta ANSI do terminal e indicadores de Git derivados da identidade visual da ExFree.
- Acento de marca na activity bar.

### Conhecido
- Atributos do C# (`[Obsolete]`) podem não receber destaque, dependendo da extensão instalada — ver `docs/roadmap.md`.
- Entidades XML (`&amp;`, `&lt;`) ainda sem regra dedicada.
