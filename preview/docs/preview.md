## Python

```python
class User:

    def __init__(self, name):

        self.name = name

    def greet(self):

        print(f"Hello {self.name}")
```

---

## Rust

```rust
struct User {

    name: String,

}

impl User {

    fn greet(&self) {

        println!("Hello {}", self.name);

    }

}
```
